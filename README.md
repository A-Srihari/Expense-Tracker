<h2>About this project</h2>
<p>A full-stack Expense Tracker web application built using Spring Boot, MySQL, and vanilla frontend technologies (HTML, CSS, JavaScript).
This project allows users to register, log in, and manage their daily expenses with full CRUD functionality.</p>

<h2>Features</h2>
<li>User Login & Signup</li>
<li>Add new expenses with amount, description, and date</li>
<li>Edit & Delete existing expenses</li>
<li>View total expenses dynamically</li>
<li>Fully responsive UI (mobile-friendly)</li>
